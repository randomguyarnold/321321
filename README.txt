-------------------------------------------------------------------------------------
Install this on any computer craft computers you want by adding these included items:
-------------------------------------------------------------------------------------
1. the "play" file
2. the "songs" folder ( add any .nbs song files you want avaialble on the computer inside this folder )
3. (optional) the "playAPIDemo" file (see video for details!)

into each cc computer folder:

\minecraft\saves\worldName\computer\[computer number]\




-------------------------------------------------------------------------------------
To add songs to the rom so they can be played on any computer, add them here:
-------------------------------------------------------------------------------------
\minecraft\mods\ComputerCraft\lua\rom\songs

NOTE:
The "ComputerCraft" folder inside \mods\ it is not there by default, you'd have to create it.
